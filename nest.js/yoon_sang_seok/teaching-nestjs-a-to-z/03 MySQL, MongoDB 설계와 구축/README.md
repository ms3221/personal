# DBMS (Dtabase Management System)

### 데이터베이스 관리 시스템

> 데이터를 어떻게 저장하고 어떻게 관리하느냐에 따라 여러가지로 분류할 수 있다

- RDBMS (Relational DBMS)

  - 관계형 데이터를 관리하는 DBMS
  - `SQL : RDBMS에서 데이터를 관리하기 위해 설계된 언어`
  - MySQL, PostgreSQL

- NoSQL (Not Only SQL)
  - SQL을 사용하지 않는 DBMS
  - Redis, MongoDB, ...
